# Docs

This directory contains GSE telemetry specification documents.

## Contents

- `GSE_Telemetry_Requirements_v0.1.docx` — Primary telemetry requirements document (upload pending)
- `requirements/` — Directory for future requirement documents

## Uploading the Requirements Document

To add `GSE_Telemetry_Requirements_v0.1.docx`:

1. Navigate to this `docs/` folder on GitHub
2. Click **Add file** → **Upload files**
3. Select the `.docx` file from your device
4. Commit the upload
