# GSE Telemetry Spec

Specifications and requirements for Ground Support Equipment (GSE) telemetry systems.

## Overview

This repository contains the telemetry requirements, standards, and specification documents for GSE systems. It serves as the central source of truth for telemetry data formats, communication protocols, and system integration requirements.

## Repository Structure

```
.
├── README.md
├── CHANGELOG.md
└── docs/
    ├── GSE_Telemetry_Requirements_v0.1.docx   # Primary requirements document
    └── requirements/                           # Future requirement documents
```

## Current Version

**v0.1** — Initial release of GSE telemetry requirements.

## Adding New Requirements

Place new requirement documents in the `docs/requirements/` directory. Follow the naming convention:

```
<Category>_<DocumentName>_v<Major>.<Minor>.<ext>
```

Update the CHANGELOG.md when adding or revising documents.
